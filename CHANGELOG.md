# 变更日志

所有重要的变更都会记录在这个文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 重构 - 2025-12-13

#### 新增
- 创建 `internal/utils` 包，提供统一的工具函数
- 添加 `types.go` 文件，集中管理共享类型定义
- 创建 `docs/` 目录，包含 API、架构和贡献指南文档
- 添加统一的随机数生成器（线程安全）
- 创建 `bin/` 目录用于存放编译后的二进制文件

#### 变更
- **重大重构**：使用标准库替代所有自定义字符串操作函数
- **性能优化**：统一随机数生成器，避免重复初始化
- **代码简化**：移除重复的工具函数（`indexOf`、`contains`、`toLower` 等）
- **结构优化**：移动 `examples/basic.go` 到 `examples/basic/main.go`
- **改进文档**：创建详细的 API 文档和架构文档

#### 修复
- 修复多个随机数生成器可能产生相同种子的问题
- 优化线程安全性
- 减少不必要的内存分配

#### 移除
- 删除重复的字符串操作函数
- 清理未使用的代码

### 技术债务清理
- 统一代码风格
- 改进错误处理
- 优化 API 设计

## [1.0.1] - 2024

### 新增
- 添加 HTTP/3 兼容说明
- 完善 User-Agent 生成功能

### 修复

## [1.0.0] - 2024

### 新增
- 初始版本
- 支持 66 个浏览器指纹（44 个主流浏览器 + 22 个移动端/自定义）
- 支持 Chrome、Firefox、Safari、Opera 等主流浏览器
- 自动生成匹配的 User-Agent
- 支持 30+ 种全球语言
- 完整的 HTTP/2 配置
- 操作系统随机化功能
